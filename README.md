# Ecoride - Dossier Projet

## Description du Projet

Ecoride est une plateforme de covoiturage écologique visant à promouvoir une mobilité durable et à réduire l'empreinte carbone des déplacements quotidiens.

## Structure du Dossier

Ce dépôt contient le dossier projet complet pour Ecoride :

- [DOSSIER_PROJET.md](./DOSSIER_PROJET.md) - Document principal du dossier projet
- [SPECIFICATIONS_TECHNIQUES.md](./SPECIFICATIONS_TECHNIQUES.md) - Spécifications techniques détaillées
- [PLANNING.md](./PLANNING.md) - Planning et jalons du projet

## Objectifs Principaux

1. Faciliter le covoiturage entre particuliers
2. Réduire l'impact environnemental des déplacements
3. Créer une communauté d'utilisateurs éco-responsables
4. Proposer une solution économique et conviviale
